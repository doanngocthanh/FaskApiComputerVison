API Endpoints

POST /api/v1/vietnam-citizens-card-ocr
Extract text from Vietnamese citizen ID cards with high accuracy. Supports both Vietnamese and English text recognition.
Request Parameters
image string
Base64 encoded image data (required)
lang string
Language code: "vi" for Vietnamese, "en" for English (default: "vi")
Request Example
{ "image": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ...", "lang": "vi" }
Success Response (200)
{ "success": true, "data": { "name": "NGUYỄN VĂN MINH", "id": "001234567890", "birth": "15/08/1990", "address": "Hà Nội, Việt Nam" }, "processing_time": 0.045 }

POST /api/v1/vietnam-citizens-card-mrz-extraction
Extract and parse Machine Readable Zone (MRZ) data from Vietnamese ID cards.
Request Parameters
image string
Base64 encoded image data (required)
Success Response
{ "success": true, "mrz_data": { "document_type": "ID", "country_code": "VNM", "document_number": "001234567890" } }
POST /api/v1/vietnam-citizens-card-detection
Detect and locate Vietnamese citizen ID cards in images with bounding box coordinates.
Request Parameters
image string
Base64 encoded image data (required)
Success Response
{ "success": true, "detections": [ { "confidence": 0.98, "bbox": [120, 50, 480, 300] } ] }
